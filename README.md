# PluseCare 
**Course:** DCIT 208  
**Team Name:** TechCare Team   
**Project Tagline:** Connecting patients to better care   

## Project Overview
PluseCare is a web application built for Dr. Awuku Samuel at the Akuapem North Community Health Center. The system bridges the gap between the hospital and its physical pharmacy by allowing patients to reach health personnel via phone calls or text messages for minor issues instead of joining long queues. 

### Key Features
* **Drug Ordering:** Patients can order basic medication from the comfort of their homes.
* **Prescription Verification:** Allows uploading photos of medical receipts for verification upon drug delivery.
* **Remote Consultation Lines:** Contains dedicated phone lines linking patients directly to either the hospital or pharmacy after a consultation fee is processed.
* **Integrated Payments:** A built-in payment portal for handling consultation and service fees seamlessly.

## Core User Groups
* **Administrator:** Responsible for managing the system backend.
* **Staff:** Tasked with updating medical and transaction records.
* **Patients:** Can access core healthcare and pharmacy services remotely.
